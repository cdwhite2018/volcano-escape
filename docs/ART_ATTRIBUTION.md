# Art and audio attribution

All runtime visuals and audio are original and local to this repository.

- The cinematic/title landscape was generated specifically for Volcano Escape from an original prompt describing a cinematic 16-bit Hawaiian sunset, a tour helicopter, ocean coastline, and a vast active volcano. It was then integrated locally without hotlinking.
- Gameplay sprites, terrain, particles, lighting, helicopter details, controls, and UI are drawn by project-owned HTML/CSS/Canvas code.
- Sound effects are synthesized at runtime with the browser Web Audio API.

No external libraries, fonts, images, audio files, CDNs, analytics, or network requests are used at runtime.
